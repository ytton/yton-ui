## API

列表的属性说明如下：

| 属性 | 说明 | 类型 | 默认值 | 版本 |
| --- | --- | --- | --- | --- |
| classNames | 语义化结构 class | [Record\<SemanticDOM, string>](#semantic-dom) | - | - |
| value | 当前数，受控值 | number | - | - |
| defaultValue | 默认值 | number | 0 | - |
| onChange | 每个item是否展示下边框 | boolean | false | - |
| disabled | 禁用，开启后变成只读效果 | Boolean | false | - |
| size | 图标大小 | 'small' \| 'meduim' \| 'large' | 'meduim' | - |
| count | 展示多少个星 | number | 5 | - |
| half | 是否允许半选 | Boolean | false | - |
