## 何时使用

最基础的列表展示，可承载文字、列表、图片、段落，常用于后台数据展示页面。

## API

列表的属性说明如下：

| 属性 | 说明 | 类型 | 默认值 | 版本 |
| --- | --- | --- | --- | --- |
| classNames | 语义化结构 class | [Record\<SemanticDOM, string>](#semantic-dom) | - | - |
| items | 列表数据 | Array\<any> | - | - |
| columns | 列数 | number | 1 | - |
| bordered | 每个item是否展示下边框 | boolean | false | - |
| renderItem | 自定义Item渲染过程 | (item) => ReactNode | (item) => item | - |
